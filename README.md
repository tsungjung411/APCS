## 關於「程式設計觀念題」
 - 出題者是很糟糕的人，變數都用 a, b, c, n, i, ... 在命名
   <br>這在業界根本不會有人寫這樣的 code
   <br>基本上，Coding conventions 可以說是「零分」
   <br>寫出的程式讓人看不懂，就是「零分」
