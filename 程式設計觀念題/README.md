## 快速掃讀技巧
- ### 技巧1
  ```C
  for (int i=1; i<=n; i=i+1)
  ```
  - 只要看到這種程式碼，用途就是
    - **foreach 每個元素**

- ### 技巧2
  ```
  # 取自 2016-03-05_觀念題 Q8
  (A) (a) f[i]=f[i-1]+f[i-2] (b) f[N]
  (B) (a) a = a + b (b) a
  (C) (a) b = a + b (b) b
  (D) (a) f[i]=f[i-1]+f[i-2] (b) f[i]
  ```
  - 如果很確定 (a) b = a + b，就不要再看 (b) 了
    <br>省下寶貴時間，有助於後面的解題
  - 此題的(b)，也沒有說明很清楚
    <br>若鑽研(b)，會耽誤寶貴時間
    
- ### 技巧3
  - 答案選項的差，**只相差1**，通常就是二擇一，其他選項就不用看了
  - (或是4個選項分兩類，每一類在二擇一)
  - 範例：
    - 2016-03-05_觀念題 Q09
      - **(B) 4** (C) 3
    - 2016-03-05_觀念題 Q10
      - (B) 18 **(C) 19**
    - 2016-03-05_觀念題 Q15
      - **(A) 16** (D) 15
    - 2016-03-05_觀念題 Q16
      - (C) 15 **(D) 16**
    - 2016-03-05_觀念題 Q24
      - **(C) 13** (D) 14
   - 如果這題太難、或是時間不夠，就用猜的，**命中率50%**

- ### 技巧4
  - 有時候，小資料用列舉的，會比計算公式快上很多
  - 範例：
    - 2016-03-05_觀念題 Q19
      ```
      A[0][0] = 108
      A[0][1] = 110
      A[0][2] = 112
      A[0][3] = 114
      A[1][0] = 116
      A[1][1] = 118
      A[1][2] = 120
      ```
